Forge - Track Habits & Streaks

Build consistent habits with a simple, offline habit tracker.

Struggling to stay consistent?
Forge is a simple habit tracker built for discipline, not motivation, not gimmicks.

No accounts. No ads. No notifications.
Just you, your habits, and consistency over time.

What Forge helps you do:
- Forge helps you track daily habits without distractions or complicated systems. It’s designed to be fast, minimal, and reliable, so you can focus on doing the work.

Key features:
Create unlimited habits
Track habit streaks
- View completion percentages for last 7, 30, 100 and 365 days.
- GitHub-style contribution grid for each habit (Each box represents a day you completed your habit)
- Calendar history view (past week)
- See overall habit completion percentage
- Create or edit habits in seconds

Powerful home screen widgets
Forge includes 3 completely free widgets:

1. Completion percentage (last 30 days)
2. Current habit streak
3. Contribution grid widget

No paywalls. No subscriptions just to use widgets.
